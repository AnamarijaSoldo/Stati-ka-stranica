SPARK School

Tko smo?
SPARK je hub za IT obrazovanje u Mostaru, posvećen studentima i digitalnim profesionalcima. Cilj nam je pružiti kvalitetno informatičko obrazovanje i na taj način povećati konkurentnost bh. tržišta. To radimo kroz naše programe - SPARK school, SPARK events i SPARK projects. SPARK tim se sastoji od visokoobrazovanih stručnjaka iz područja poput digitalnog marketinga, organiziranja događaja, edukacije, financija i upravljanja projektima. SPARK je razvio široku mrežu partnerstava s vodećim korporativnim tvrtkama, kao što su Sportradar i NSoft.
Koje su najaktivnije radionice?

Web programiranje
Programiranje je davanje pisanih instrukcija računalu na logičan način, koji računalo može razumijeti. Računalu dajete instrukcije u malim koracima, koje on izvršava jednu za drugom. Programiranjem stvarate novi softver i navodite računalo na nove stvari. Web programiranje vam omogućava da jednostavne, statičke HTML stranice pretvorite u dinamička master-djela.

Backend
Backend developeri stvaraju, kodiraju i poboljšavaju servere, server-side aplikacije i baze podataka, koje kad se povežu s frontend kodom, zajedno stvaraju funkcionalno, neprekidno iskustvo za krajnjeg korisnika. Jednostavno rečeno backend ""hendla"" sve one ""behind the scenes"" funkcionalnosti web aplikacija. To uključuje kod koji povezuje web i bazu podataka, upravlja korisničkim konekcijama i osnažuje web aplikaciju.

Frontend
Frontend development je poznat kao klijent-strana developmenta i uključuje izradu HTML-a, CSS-a i JavaScript-a za web stranice ili web aplikacije kako bi ih korisnik mogao vidjeti i komunicirati s njima. Frontend developediv ri grade i razvijaju web stranice i aplikacije koristeći različite web tehnologije, analiziraju kod, dizajniraju i debuggiraju aplikacije s ciljem besprijekornog korisničkog iskustva.
